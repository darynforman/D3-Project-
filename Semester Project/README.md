# Belize Rainfall Visualization

An interactive data visualization project that displays monthly rainfall data across different districts in Belize. Built with D3.js, this web application provides a clear and engaging way to explore rainfall patterns throughout the year.


## Features

- **Interactive Line Chart**: Visualizes monthly rainfall data for multiple districts
- **Responsive Design**: Adapts to different screen sizes
- **Data Comparison**: Easily compare rainfall patterns between different districts
- **Smooth Animations**: Engaging transitions when interacting with the data
- **Interactive Tooltips**: Hover over data points to see exact values

## Project Structure

- `index.html` - Main HTML file with the application structure
- `style.css` - Styling for the visualization
- `script.js` - D3.js code for the interactive visualization
- `data.csv` - Rainfall data for different districts in Belize
- `README.md` - This documentation file

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- No server required - runs directly in the browser

### Running the Application

1. Clone this repository or download the source files
2. Open `index.html` in your preferred web browser
3. Interact with the visualization by hovering over data points

## Data Source

The application uses rainfall data for different districts in Belize. The data is stored in `data.csv` with the following format:
- Each row represents a district
- Columns represent months (Jan, Feb, ..., Dec)
- Values represent rainfall measurements

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- [D3.js](https://d3js.org/) - Data visualization library
- Google Fonts (Inter)

## Customization

You can easily customize this visualization by:
1. Updating the data in `data.csv`
2. Modifying the color scheme in `script.js`
3. Adjusting the chart dimensions in both `script.js` and `style.css`
